Welcome in yourFinance program.

This program helps you to manage your budget. It saves budget data as separate money deposition places for certain month and certain year. You can check detailed data as well as analyze certain month to see the values and other information.

What would you like to do:

  • Add new data
  • View, edit and delete data
  • Delete multiple data
  • Analyze data:
    • Analyze last month
    • Analyze chosen month
  • Configure settings:
    • Configure names of money deposition places
    • Configure monthly costs names and amounts
    • Configure current costs groups
  • Exit
